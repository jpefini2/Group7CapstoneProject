Clear Notifications:

Context: The user has at least one notification

Primary Flow:
1. The user selects the notification pane.
2. The user selects the "x" next to the notification.
3. The system deletes the notification from the database.
4. The notification is removed from the notification pane.

Alternate Flow:
	A2. The user selects the "clear all notificaitons" button.
	A3. The system deletes the notifications from the database.
	A4. All notifications are cleared from the users pane.

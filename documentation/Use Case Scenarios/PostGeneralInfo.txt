Post General Information:

Context: The advisor is on the main screen.

Primary Flow:
1. The user selects the post button.
2. The system informs the user to enter the information.
3. The user enters the information.
4. The user selects confirm.
5. The system persists the informatino to the database.
6. The system reloads the page with the new information.

Alternate Flow:
	A4. The user hits cancel.
	A5. The user is taken back to the previous screen.
	
Alternate Flow:
	A5. The system fails to persist the information.
	A6. The system informs the user of the error.
	A7. Return to primary flow 2.

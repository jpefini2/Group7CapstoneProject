Create Profiles:

Context: The student is at the accounts page.

Primary Flow:
1. The user selects create/update profile.
2. The user enters the required information.
3. The user hits confrim
4. The system persists the information.
5. The system reloads the page with the new information.

Alternate Flow: 
	A4. The system fails to persist the information.
	A5. The system notifies the user of the error.
	A6. Return to primary flow 2.

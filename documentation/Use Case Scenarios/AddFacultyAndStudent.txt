Add Faculty and Students to System:

Context: The general advisor is at the "add user page"

Primary Flow:
1. The user selects "add new"
2. The system prompts the user to selects student or advisor.
3. The user selects desired user type.
4. The user enters the required information.
5. The user selects add.
6. The new user is added to the system.
7. The system notifies the user of successful add.

Alternate Flow:
	A3. The user selects cancel.
	A4. The system returns to the previous page.
	
Alternate Flow:
	A6. The system could not add the new user.
	A7. The system notifies the user of the reason add was not successful.
	A7. return to primary flow 4.

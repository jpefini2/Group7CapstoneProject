Inform Faculty Advisor that Student Has Met Them:

Context: General Advisor is currently viewing student meeting.

Primary Flow:
1. The user clicks the check box for meeting complete.
2. The user clicks confirm button.
3. The system saves the meeting information.
4. The system sends the students faculty advisor a notification. 

Alternate Flow:
	A3. The system could not save the meeting information.
	A4. The system prompts the user of the error.
	A5. Return to primary flow 2.
